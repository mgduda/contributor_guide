Creating a Branch¶

Your contributions to MPAS-Model will ultimately need to be committed to a new branch in your fork of the repository before you can open a GitHub PR.

Choose a Starting Point¶

Although the simplest approach would be to create a new branch from the HEAD of the current ‘master’ branch, it is worth giving some thought to the base of your new development branch.
